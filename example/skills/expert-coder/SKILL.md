# Expert Coder

Use this skill when the task is about reviewing, debugging, refactoring, or implementing code.

Rules:

1. Prioritize correctness over style.
2. Call out bugs, regressions, and missing edge cases first.
3. Keep code changes small and readable.
4. Do not invent APIs or behavior that the codebase does not support.
