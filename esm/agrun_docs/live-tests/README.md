## How To Use This Index

- **Verifying an ADR:** Find the ADR section above; live-test records prove the contract held in real LLM runs.
- **Adding a new live test:** Drop the `<topic>-YYYY-MM-DD.md` file in this directory and add an entry to the matching section above.
- **Stale records:** Live tests are not maintained as architecture rotates. If a record contradicts a newer ADR, the ADR wins (per `architecture-ssot.md` authority order).
